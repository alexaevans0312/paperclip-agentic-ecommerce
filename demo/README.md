# Demo Video Source

This directory contains the planned demo package for the Paperclip Agentic Ecommerce hackathon submission.

The demo should feel like an operating-system walkthrough, not a product pitch deck. It needs to show a real founder goal becoming a governed ecommerce workflow.

## Files

- `FRAME.md` — creative frame: objective, audience, narrative, visual language.
- `storyboard.md` — 1-3 minute scene-by-scene structure.
- `narration-script.md` — draft voiceover script for the X demo video.
- `shot-list.md` — exact screens/artifacts to capture.

## Demo target

Runtime: 90-150 seconds.

Core arc:

```text
messy founder growth goal
  -> Paperclip operator layer
  -> specialist ecommerce agents
  -> Stripe approval gates
  -> Nemotron review
  -> founder-ready operating packet
```

## Production notes

- Keep the language concrete: offer, CAC, AOV, claims, spend cap, approval.
- Do not overclaim live payment execution unless it has been verified.
- Show the repo, docs, example brief, approval ledger, and operating packet.
- End with the idea that Squish is the first use case for a reusable agentic ecommerce pattern.
